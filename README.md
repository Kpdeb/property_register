# 🏠 Property Register (Soroban Smart Contract)

## 📌 Project Description

**Property Register** is a decentralized smart contract built on the Stellar Soroban platform that enables secure, transparent, and immutable property ownership management. This system eliminates the need for centralized authorities by leveraging blockchain technology to record and verify property ownership.

---

## ⚙️ What It Does

The Property Register smart contract allows users to:

- Register new properties on-chain  
- Assign ownership to a specific wallet address  
- Transfer property ownership securely  
- Verify property ownership instantly  
- Maintain a tamper-proof record of all property transactions  

All operations are executed through the Soroban smart contract, ensuring trust, transparency, and decentralization.

---

## 🚀 Features

- 🔐 **Decentralized Ownership**  
  No reliance on central authorities—ownership is managed via blockchain.

- 📝 **Property Registration**  
  Users can register properties with unique identifiers.

- 🔄 **Ownership Transfer**  
  Secure and transparent transfer of property ownership between users.

- 🔍 **Ownership Verification**  
  Anyone can verify the owner of a property in real time.

- 📜 **Immutable Records**  
  Once recorded, property data cannot be altered or deleted.

- ⚡ **Fast & Low-Cost Transactions**  
  Powered by Stellar’s efficient network.
